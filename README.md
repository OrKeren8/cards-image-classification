# cards-image-classification

This repository is set up for Jupyter notebook usage in image classification projects.

## Setup

1. Ensure Python is installed.
2. Create and activate virtual environment: `python -m venv .venv` and activate it.
3. Install dependencies: `pip install -r requirements.txt`
4. Launch Jupyter: `jupyter notebook`

## Dataset

The dataset is located in the `../cards_dataset/` folder with train, test, and valid splits.
